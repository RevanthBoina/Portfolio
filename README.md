# Rahul Lakhaney — Dubai Luxury Parallax Website

A fully functional standalone HTML/CSS/JS cinematic parallax website using a scrollable frame box instead of normal page scrolling.

## Main file

Open:

```text
index.html
```

## Assets

All 24 generated Dubai luxury images are included inside:

```text
assets/dubai_24_shots/
```

## Features

- Body scroll disabled
- Internal frame-box scrolling only
- Smooth Lenis-like scroll implemented in vanilla JavaScript
- 24 cinematic sections
- Dubai 7-star luxury hotel-office image backgrounds
- Parallax image movement
- Animated counters
- Progress bar
- Chapter navigation
- Chapter indicator
- Keyboard controls: Arrow Up/Down, Page Up/Down, Space, Home, End
- Responsive mobile/tablet styling
- Independent concept disclaimer
